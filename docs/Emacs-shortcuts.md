`C-x k` (`kill-buffer`) kills one buffer, whose name you specify in the minibuffer.\
`C-x 0` - to delete the current window


# Dired shortcuts
`g` - reload buffer\
`m` - mark a file or subdirectory for later commands\
`C` - copy file(s)\
`R` - rename a file or move files to another directory\
`L` - run elisp script
