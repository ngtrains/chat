# Chat project
## Technologies:
  Docker - runs project in containers.\
  Emacs - code editor. It is optional.\
  [TMUX](docs/TMUX.md) -  is a terminal multiplexer. It lets you switch between programs in one terminal, detach them and reattach them to a different terminal.\
  Git - distributed version control system.
